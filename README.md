# App-Antrian
Merupakan sebuah Aplikasi Antrian yang di Develop untuk sebuah instansi Pemerintah, Direktorat Metrologi.

Aplikasi ini dibuat secara terstruktur dengan sistem :
1. Kiosk App Sebagai Server dan Database dan mengambil Antrian.
2. Admin App Sebagai operator yang mengendalikan setiap Loket, memanggil, menukan, dan sebagainya.
3. Dan semua data berjalan secara real-time dan di tampilkan menggunakan Display yang di hubungkan ke TV.

Bahasa Pemrograman yang digunakan :
1. HTML
2. CSS
3. JS

Framework yang digunakan :
1. Electron
2. Node.js

Aplikasi ini dapat membantu staff Metrologi terutama di bagian Pelayanan Masyrakat agar lebih efisien dalam melakukan layanan dan pemanggilan setiap nomor antrian.
